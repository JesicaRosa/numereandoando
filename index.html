Numereando ando

Diccionario de lectura facilitada

Contar

Definición:
Decir la cantidad que hay de algo.
Ejemplo:
Voy a contar cuántos bocadillos hay en la mochila.

matemáTIClab

Definición:
Es el nombre de un proyecto con el que se hacen materiales que ayudan a las personas aprender a contar.
Ejemplo:
Este proyecto se llama matemáTIClab y es muy original.

Números

Definición:
Signo que representa una cantidad. Por ejemplo: 1, 2, 3.
Ejemplo:
Esta niña ya sabe los números.
